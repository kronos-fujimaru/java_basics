## 演習18（ex18.java）

```java
class ex18 {
    public static void main(String[] args) {
        int[] scores = {65, 23, 47, 92, 53};

        for (int i = 1; i <= scores.length; i++) {
            for (int j = 0; j < scores.length - i; j++) {
                if (scores[j] > scores[j + 1]) {
                    int temp = scores[j + 1];
                    scores[j + 1] = scores[j];
                    scores[j] = temp;
                }
            }
        }

        for (int score : scores) {
            System.out.println(score);
        }
    }
}
```
